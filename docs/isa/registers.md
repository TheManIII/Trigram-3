# Trigram-3 寄存器规范

## 寄存器概览

Trigram-3 有 9 个寄存器（R0-R8），每个寄存器宽度为 9 trit。

## 寄存器列表

| 寄存器 | 中文名 | 编码（3 trit） | 十进制 | 说明 |
|--------|--------|---------------|--------|------|
| R0 | 零号 | 000 | 0 | 硬连线为 0 |
| R1 | 一号 | 001 | 1 | 通用寄存器 |
| R2 | 二号 | 01T | 2 | 通用寄存器 |
| R3 | 三号 | 010 | 3 | 通用寄存器 |
| R4 | 四号 | 011 | 4 | 通用寄存器 |
| R5 | 五号 | 1TT | 5 | 通用寄存器 |
| R6 | 六号 | 1T0 | 6 | 通用寄存器 |
| R7 | 七号 | 1T1 | 7 | 通用寄存器 |
| R8 | 八号 | 10T | 8 | 通用寄存器 |

## 寄存器编码

寄存器字段使用平衡三进制编码（T, 0, 1）：

### 编码原理

Trigram-3 使用平衡三进制对寄存器编号进行编码：

```
R0 → 000 = 0
R1 → 001 = 1
R2 → 01T = 2
R3 → 010 = 3
R4 → 011 = 4
R5 → 1TT = 5
R6 → 1T0 = 6
R7 → 1T1 = 7
R8 → 10T = 8
```

### 编码验证

```python
# 验证编码正确性
assert int("000", base=3) == 0   # R0
assert int("001", base=3) == 1   # R1
assert int("01T", base=3) == 2   # R2 (T = -1)
assert int("010", base=3) == 3   # R3
assert int("011", base=3) == 4   # R4
assert int("1TT", base=3) == 5   # R5
assert int("1T0", base=3) == 6   # R6
assert int("1T1", base=3) == 7   # R7
assert int("10T", base=3) == 8   # R8
```

## 特殊寄存器

### R0 - 零号寄存器

- **编码**: 000
- **值**: 恒为 0
- **用途**: 
  - 作为立即数加载的源寄存器
  - 提供常量 0
  - 符合 RISC 传统

### R1-R8 - 通用寄存器

- **宽度**: 9 trit
- **值域**: -9841 到 +9841
- **用途**: 
  - 存储计算结果
  - 作为指令操作数
  - 临时数据存储

## 寄存器使用约定

虽然 Trigram-3 没有强制的调用约定，但建议遵循以下约定：

| 寄存器 | 建议用途 |
|--------|----------|
| R0 | 常量 0 |
| R1 | 函数返回值 |
| R2-R3 | 函数参数 |
| R4-R6 | 临时寄存器 |
| R7-R8 | 保存寄存器 |

## 数值范围

每个寄存器可存储的数值范围：

- **最小值**: -9841 (T00000000)
- **最大值**: +9841 (100000000)
- **零值**: 0 (000000000)

## 在指令中的使用

### 作为目标寄存器 (Rd)

```assembly
ADD R1, R2, R3    ; R1 = R2 + R3
OFFSET R4, R0, #5 ; R4 = 0 + 5
```

### 作为源寄存器 (Rs1, Rs2)

```assembly
ADD R1, R2, R3    ; R2 和 R3 作为源
CMP R4, R1, R2    ; R1 和 R2 作为源
```

### 作为基址寄存器

```assembly
LOAD R3, [R1, #5]   ; R1 作为基址
STORE R4, [R2, #3]  ; R2 作为基址
```

## 寄存器溢出

Trigram-3 寄存器采用固定宽度，运算时可能发生溢出：

- **加法溢出**: 结果超出 [-9841, +9841] 范围
- **处理方式**: 结果自动截断为 9 trit
- **程序员责任**: 确保运算结果在有效范围内

## 示例

### 示例 1: 寄存器初始化

```assembly
OFFSET R1, R0, #10  ; R1 = 10
OFFSET R2, R0, #20  ; R2 = 20
OFFSET R3, R0, #30  ; R3 = 30
```

### 示例 2: 寄存器运算

```assembly
ADD R4, R1, R2      ; R4 = R1 + R2 = 30
CMP R5, R4, R3      ; R5 = sign(30 - 30) = 0
NEG R6, R1          ; R6 = -R1 = -10
ABS R7, R6          ; R7 = |R6| = 10
```

### 示例 3: 寄存器到内存

```assembly
STORE R1, [R0, #100]  ; Mem[100] = R1 = 10
LOAD R8, [R0, #100]   ; R8 = Mem[100] = 10
```
